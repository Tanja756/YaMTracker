## YaMSpy

### Описание

Расширение для браузера Chrome, предназначенное для работы с чатом Яндекс Мессенджера. Основная цель расширения — отслеживать и отображать заявки (ордера) в интерфейсе чата.

### Функционал

- **Отслеживание заявок**: автоматически обнаруживает номера заявок в сообщениях и связывает их.
- **Панель с заявками**: отображает таблицу с данными о заявках, включая дату, автора, статус, сроки.
- **Экспорт в CSV**: позволяет сохранить данные в формате CSV для дальнейшей аналитики.
- **Фильтрация по месяцам**: пользователь может сменить месяц для просмотра заявок.

### Как использовать

1. Загрузите расширение в Chrome.
2. Откройте чат Яндекс Мессенджера.
3. Нажмите на иконку расширения, чтобы открыть панель с заявками.
4. Используйте кнопки для смены месяцев и экспорта данных.
