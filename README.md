# 🔧 Yamb Orders Analyzer

Расширение для Firefox, которое анализирует заявки из чатов **Яндекс Мессенджера** (messenger.360.yandex.ru) и выводит сводную таблицу с отслеживанием закрытия.

## 📋 Возможности

- Автоматически находит все заявки по шаблонам **ХК‑XXXXXX** и **ЗНО‑XXXXXX** (объекты с упоминанием «Пятерочка»).
- Строит цепочки ответов на каждую заявку (по цитированию).
- Определяет **дату закрытия** – время последнего сообщения в цепочке.
- Показывает в удобной таблице:
  - Номер заявки
  - Код ИНЦ / ЗНО
  - Объект обслуживания
  - Дата и время заявки (фактическая отправка)
  - Дата и время закрытия
  - Скорость реакции (часы `X.XX` и минуты)
  - Текст последнего ответа (закрытия)
  - Автор, закрывший заявку
- Выделяет строки, где закрытие заняло **более 8 часов**.
- Фильтрация по месяцам с переключением стрелками ◀ ▶.
- Экспорт текущей таблицы в **CSV** (совместим с Excel).
- Работает полностью локально, не отправляет данные вовне.

## 📦 Установка

### Временная загрузка в Firefox

1. Склонируйте репозиторий или скачайте его ZIP-архивом.
2. Откройте Firefox и перейдите на страницу `about:debugging#/runtime/this-firefox`.
3. Нажмите **«Загрузить временное дополнение»**.
4. Выберите файл `manifest.json` из папки проекта.
5. Иконка расширения появится на панели инструментов.

### Постоянная установка (через подпись)

Для постоянного использования без отладки необходимо подписать расширение в [AMO](https://addons.mozilla.org/developers/). Инструкция выходит за рамки этого README, но структура манифеста полностью готова для загрузки.

## 🚀 Использование

1. Откройте нужный чат в [Яндекс Мессенджере](https://messenger.360.yandex.ru).
2. Нажмите на иконку расширения (📋) на панели инструментов.
3. Справа появится панель с таблицей заявок.
   - **Фильтр по месяцу** – используйте стрелки влево/вправо для переключения.
   - **CSV** – кнопка «📥 CSV» сохраняет текущую таблицу в файл.
4. Новые сообщения в чате автоматически обновляют данные.

## 🧩 Как это работает

Расширение анализирует DOM‑дерево страницы чата:

- Ищет все сообщения (селектор `.yamb-message-row`).
- Определяет заявки по тексту без цитирования (содержат `ХК‑...` и `Пятерочка`).
- Связывает ответы с заявками по **совпадению начала нормализованного текста** (только русские буквы).
- Строит цепочку ответов и находит самое позднее сообщение — это закрытие.
- Все вычисления происходят в реальном времени при появлении новых сообщений.


## ⚠️ Примечания

- Расширение рассчитано на текущую вёрстку Яндекс Мессенджера. При значительных изменениях в DOM селекторы могут потребовать обновления.
- Данные хранятся только в памяти страницы и сбрасываются при перезагрузке чата.
- Все вычисления выполняются локально, никакая информация не передаётся на сторонние серверы.
